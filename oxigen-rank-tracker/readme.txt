== Description ==
Keyword Rank Tracker is a WordPress plugin that allows you to track the daily rankings of your keywords in search engines. With an easy-to-use interface, you can manage your keywords and monitor their performance over time.

== Installation ==
1. Upload the `keyword-rank-tracker` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the Keyword Rank Tracker settings to add your keywords and start tracking.

== Usage ==
Once activated, you can access the Keyword Rank Tracker from the WordPress admin menu. Add your keywords and view their rankings on the dashboard. The plugin will automatically update the rankings daily.
